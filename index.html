A dual-layer automated attendance system built as a B.Sc. final year project at Kongunadu Arts and Science College, developed in collaboration with Botomate Technologies, Coimbatore. The system combines facial recognition with ID card verification to completely eliminate proxy attendance — if a student's detected face doesn't match their ID card, attendance is flagged and alerts are sent instantly.

MTCNN handles real-time face detection with landmark alignment, while FaceNet extracts high-dimensional embeddings for identity matching using cosine similarity. The ID card module cross-verifies the captured face label against a Teachable Machine–trained model. All attendance records are auto-logged into timestamped Excel sheets and emailed to staff with full Present / Late / Absent breakdowns.

CropGuard AI is a comprehensive hybrid multi-model crop disease detection system evaluated on 1,803 leaf images spanning 13 classes across four major crops: Apple, Pepper, Potato, and Tomato. The system employs MobileNetV2 as a CNN feature extractor (1,280-dimensional embeddings), combined with classical ML classifiers — SVM, Random Forest, and Logistic Regression — through a soft-voting ensemble strategy.

11 model configurations were systematically evaluated: 4 individual classifiers, 6 pairwise hybrids, and 1 full ensemble. The CNN+SVM hybrid achieved the best performance, statistically validated via Friedman test and Wilcoxon signed-rank test. The system is deployed as a Streamlit web application featuring real-time disease detection, a 5-level severity estimator, and AI-powered treatment recommendations via the Gemini API.
